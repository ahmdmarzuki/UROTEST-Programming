pull terlebih dahulu folder dengan nama "Robot Battle Game" dari GitHub ke computer.

buka folder tersebut di vscode atau code editor lain.

jalankan(run) file "main.py" di code editor.

pada scene pertama akan menampilkan opsi robot yang dapat dipilih dimana terdapat 3 opsi robot.

Setiap robot memiliki kelebihan dan kekurangan masing masing.
Robot 1 = 80 hp, 8 dmg
Robot 2 = 100 hp, 6 dmg
Robot 3 = 120 hp, 4 dmg

setelah memilih player 1 dan 2, tekan tombol "fight" untuk ke scene "Battle".

Kontrol player 1:
A = Move Left
D = Move Right
W = Jump
Left Click mouse = Attack

Kontrol player 2:
LEFT = Move Left
RIGHT = Move Right
UP = Jump
DOWN = Attack

untuk attack, terdapat rectangle hijau didepan player Ketika menekan tombol attack. Ketika target(musuh) mengenai rectangle tersebut, otomatis akan mengurangi current_health nya.
